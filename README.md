GLSLTester
==========

GLSL shader experimentation testbed & model viewer, written in C#.

![GLSLTester](http://i.imgur.com/n9ATQ5s.png)

Requirements
============

* .NET Framework 4
* OpenTK 1.1 (currently uses stable-3, 2014-05-19)
* Nini 1.1.0
* FastColoredTextBox
* Cyotek.Windows.Forms.ColorPicker
* Aglex
